Coco coir es básicamente la reutilización de
residuos de coco para crear soil.
Es biodegradable y perfecto para Microgreens.

Dado que los nutrientes para los microgreens solo
provienen del mismo seed, el agua y la fotosintesis,
el soil no es tan importante.
Esto es gracias a la **endosperm reserves**.
En la primera fase, no se necesita soil para los nutrientes.
El soil es simplemente un **anchor y delivery system** para el agua.

Gracias a la capacidad de aireado del coco coir, (**macropores**)
el oxigeno puede pasar perfectamente entre los seeds,
y crear el ecosistema perfecto para que las microgreens crezcan.

El Coco Coir genera canales de aire que se quedan abiertos,
incluso cuando el medio está saturado de seeds o elementos.

Tiene una buena capacidad de absorción y retención de agua.
La humedad necesaria es perfecta para ellas.
